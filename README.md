# WaterApp - Track and Reduce Your Water Footprint

WaterApp is a mobile application designed to help users track and reduce their water footprint, contributing to a more sustainable future. Through interactive surveys, challenges, and achievements, users can understand their water consumption patterns and take meaningful steps to reduce their environmental impact.

## ✨ Features

- **Initial Survey**: Complete a comprehensive 10-question survey to determine your initial water footprint
- **Personalized Challenges**: Receive tailored challenges based on your survey responses
- **Achievement System**: Track your progress and earn achievements as you reduce your water footprint
- **Educational Videos**: Watch informative videos to learn about water conservation
- **Progress Tracking**: Monitor your water footprint reduction over time
- **Daily Notifications**: Get morning and evening reminders for pending tasks
- **Profile Dashboard**: View your water conservation journey and achievements

## 🛠 Technical Details

### Built With
- React Native
- Context API for state management
- AsyncStorage for local data persistence
- Custom notification system
- RESTful API integration

### Key Components
- Authentication system with JWT
- Real-time water footprint calculation
- Video tutorial integration
- Multi-language support (English)
- Tab-based navigation
- Custom UI components

### API Integration
The app communicates with a backend server for:
- User authentication
- Water footprint profile management
- Progress tracking
- Challenge completion verification

### Data Management
- Secure local storage for user preferences
- Real-time synchronization with backend
- Offline capability for core features
- Efficient state management using Context API

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or later)
- React Native development environment
- iOS development environment (for iOS builds)
- Android development environment (for Android builds)

### Installation

1. Clone the repository:
\`\`\`bash
git clone [repository-url]
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
# or
yarn install
\`\`\`

3. Install iOS dependencies:
\`\`\`bash
cd ios && pod install && cd ..
\`\`\`

4. Start the development server:
\`\`\`bash
npm start
# or
yarn start
\`\`\`

5. Run the app:
\`\`\`bash
# For iOS
npm run ios
# or
yarn ios

# For Android
npm run android
# or
yarn android
\`\`\`
